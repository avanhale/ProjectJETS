------------------------------------
8K Skyboxes - Starfield & Nebula
------------------------------------

Contains:
- 6 Cubemap Textures
- 6 Skybox Materials

Texture size:
8192 x 4096 pixel 

To use the skyboxes, please open the light settings window and replace the skybox material.

Please note:
* On Android platform, the maximum allowed cubemap size is 2048.
* On iOS platform, the maximum allowed cubemap size is 4096.
* In this package, the "max size" attribute of all cubmap textures is set to 4096 by default. If you want to use the skyboxes on Android platform, please change the "max size" attribute of the cubemap textures to 2048.
